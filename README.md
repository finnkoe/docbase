# DocBase

DocBase is a local document and knowledge management system written in C++.

The goal of the project is to build a modular application that can import, analyze, store, and search documents locally without relying on external cloud services.

The project is designed as a long-term software engineering project with a strong focus on:

* clean architecture
* modular design
* maintainability
* local-first storage
* extensibility
* performant search

## Goals

The application should be able to:

* import text-based documents
* analyze document contents
* store documents and metadata
* provide full text search
* support future intelligent search features

## MVP Features

The first version of the project should include:

* TXT file import
* Markdown file import
* SQLite-based storage
* full text search
* command line interface
* modular software structure

## Planned Future Features

Possible future extensions:

* semantic search
* embeddings
* PDF support
* related document detection
* intelligent document analysis

## Technical Goals

This project is also intended as a software engineering and architecture project.

Focus areas:

* modern C++
* modular architecture
* separation of concerns
* database integration
* testability
* maintainable code structure
